# 🩺 Diabetes Predictive System

An end-to-end Machine Learning web application that predicts the likelihood of diabetes based on clinical diagnostic metrics. The system utilizes a Support Vector Machine (SVM) classifier for predictions, wrapped in an interactive user interface built with Streamlit.

## 📌 Project Overview
Early detection of diabetes can significantly improve patient outcomes. This project bridges the gap between machine learning pipelines and real-world utility by providing healthcare professionals or individuals with an accessible, real-time diagnostic screening tool.

### Key Features:
* **Interactive UI:** Smooth numeric inputs for diagnostic health indicators.
* **Instant Risk Assessment:** Clear visual indicators indicating whether the patient is predicted as diabetic or non-diabetic.
* **Robust Pipeline:** Data is automatically standardized before being processed by the predictive engine to maintain accurate classification performance.

## 🛠️ Tech Stack
* **Frontend/UI:** Streamlit
* **Machine Learning:** Scikit-learn
* **Data Processing:** NumPy, Pandas
* **Model Pipeline:** Support Vector Machine (SVM) Classifier & StandardScaler
